# Лабораторная работа №4 — Безопасный обмен данными по сети (RSA)

Клиент-серверное приложение на C# с асимметричным шифрованием RSA.
Каждое сообщение шифруется открытым ключом получателя и расшифровывается его закрытым ключом.

## Как это работает

```
Клиент                          Сервер
  |                               |
  |--- открытый ключ клиента ---->|   (шаг 2)
  |<-- открытый ключ сервера -----|   (шаг 3)
  |                               |
  |--- зашифровано ключом сервера>|   (шаг 4)
  |        (сообщение пользователя)
  |                               |   расшифровка закрытым ключом сервера
  |<-- зашифровано ключом клиента-|   (шаг 5)
  |        ("ваше сообщение доставлено")
  |                               |
  |   расшифровка и вывод         |   (шаг 6)
  |   отключение                  |
```

## Требования

- [.NET 10 SDK](https://dotnet.microsoft.com/download) (или выше)
- Windows / Linux / macOS

## Структура проекта

```
Sokolov LR/
```

## Запуск

### 1. Сборка

Из корневой папки проекта:

```bash
dotnet build
```

Или собрать каждый проект отдельно:

```bash
dotnet build RsaServer/RsaServer.csproj
dotnet build RsaClient/RsaClient.csproj
```

### 2. Запуск сервера

Открыть **первый терминал** и выполнить:

```bash
dotnet run --project RsaServer
```

Сервер запустится и будет ждать подключений на порту **8006**.

```
Сервер: ключи сгенерированы.
Сервер запущен. Ожидание подключений...
```

### 3. Запуск клиента

Открыть **второй терминал** и выполнить:

```bash
dotnet run --project RsaClient
```

```
Клиент: ключи сгенерированы.
Открытый ключ клиента отправлен.
Получен открытый ключ сервера.
Введите сообщение: Привет, сервер!
Ответ сервера: ваше сообщение доставлено
```

> Сначала запускайте сервер, затем клиент.

## Используемые библиотеки (.NET)

| Пространство имён | Что используется |
|---|---|
| `System.Net.Sockets` | `Socket`, `IPEndPoint` — TCP-соединение |
| `System.Security.Cryptography` | `RSACryptoServiceProvider` — генерация ключей, шифрование/расшифровка RSA |
| `System.Text` | `Encoding.UTF8` — конвертация строк в байты |

Все библиотеки входят в стандартный .NET SDK — дополнительных пакетов устанавливать не нужно.

## Параметры по умолчанию

| Параметр | Значение |
|---|---|
| Адрес сервера | `127.0.0.1` |
| Порт | `8006` |
| Размер RSA-ключа | `2048` бит |
| Паддинг | OAEP |
